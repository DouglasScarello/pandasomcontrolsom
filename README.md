# 🐼 PandaControl - Volume Center v1.0

![PandaControl Logo](https://github.com/DouglasScarello/pandasomcontrolsom/raw/main/PandaControl_Release/icon.png)

O **PandaControl** é um controlador de áudio desktop premium e divertido, desenvolvido especialmente para resolver o problema de reguladores de som físicos quebrados com estilo e fofura.

## ✨ Funcionalidades

- 🎨 **Design Glassmorphism**: Interface moderna e transparente.
- 🐼 **Orelhas Reativas**: As orelhas do Panda crescem conforme você aumenta o volume!
- 🔴 **Modo Overdrive**: Suporte para volumes de até 150% no Linux (o anel fica vermelho!).
- 📦 **Instalador de Um Clique**: Simples para quem não é programador.
- 🖥️ **Desktop Nativo**: Roda de forma independente (Electron + React).

## 🚀 Como Instalar (Linux)

1. Baixe o arquivo [PandaControl_Linux.zip](https://github.com/DouglasScarello/pandasomcontrolsom/raw/main/PandaControl_Linux.zip).
2. Extraia a pasta.
3. Clique com o botão direito no arquivo `install.sh` e escolha **Executar** (ou rode `./install.sh` no terminal).
4. Procure por **PandaControl** no seu menu de aplicativos e divirta-se!

## 🛠️ Tecnologias

- **Frontend**: React + Vite + Framer Motion
- **Core**: Electron (Node.js)
- **Áudio**: Pactl (Linux Engine)
